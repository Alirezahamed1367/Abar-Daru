# نرم‌افزار مدیریت انبار دارو

این پروژه شامل یک نرم‌افزار تحت وب برای مدیریت انبار دارو با امکانات کامل و دیتابیس SQLite است.

## ساختار پروژه
- `backend/` : بک‌اند با FastAPI و دیتابیس SQLite
- `frontend/` : فرانت‌اند با React و پشتیبانی از راست‌چین فارسی
- `db_backup/` : پوشه بکاپ خودکار دیتابیس

## امکانات کلیدی
- مدیریت انبارها و داروها
- ثبت رسید و حواله انبار
- ثبت و مشاهده تاریخچه عملیات
- گزارشات قابل خروجی PDF/Excel
- هشدار داروهای نزدیک به تاریخ انقضا
- فشرده‌سازی تصاویر دارو (حداکثر ۲ مگابایت)
- بازیابی و تغییر رمز عبور
- کنترل سطح دسترسی کاربران
- بکاپ‌گیری خودکار دیتابیس

## راه‌اندازی سریع
راهنمای نصب و اجرای پروژه پس از پیاده‌سازی کامل اضافه خواهد شد.
